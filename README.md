# Gin MVC Server
This is a simple model view controller type server with gin

## How to use?
1. **Clone the Repo**
```
$ git clone https://github.com/idkashutosh/gin-mvc-server.git
```

2. **Change to Directory**
```
$ cd gin-mvc-server
```

3. **Get the Packages**
```
$ go get ./...
```

4. **Run the Project**
```
$ go run main.go
```

**Now open you browser at http://localhost:8080 to see the result**
